STG401 - Manage objects and optimize cost with Amazon S3 and Amazon S3 Glacier In this workshop, get hands-on experience managing large amounts of data in Amazon S3. Learn best practices for configuring and managing object policies, including storage buckets, security, regulatory compliance, data replication, and more. We review optimizing storage costs, data lifecycle management, and retrieval times through the various restore tiers and SLAs, and we share strategies for moving and organizing data in formats that take advantage of Amazon S3 features. We also discuss how efficient Amazon S3 clients use the Amazon S3 API. In addition, learn about listing objects, Amazon S3 inventory, Amazon S3 batch operations, and Amazon S3 Select. STG403 - Security best practices with Amazon EFS Maintaining strict security requirements for NFS file systems while increasing innovation is top of mind for organizations today. Amazon Elastic File System (Amazon EFS) provides scalable, cloud-native, POSIX-compliant file storage for use with your Amazon Elastic Compute Cloud (Amazon EC2) Linux workloads and on-premises resources. In this chalk talk, we explore security best practices for Amazon EFS, from data encryption to permissions and managing access with AWS Identity and Access Management (IAM). By the end of the session, you have real-life examples and new insights to improve the security of your file-based business-critical applications. View Less Chalk Talk Geert Jansen Darryl Osborne
