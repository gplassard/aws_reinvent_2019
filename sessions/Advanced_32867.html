GPSTEC338 - Building data lakes for your customers with SAP on AWS Although this Global Partner Summit breakout is open to anyone, it is geared toward current and potential AWS Partner Network Partners. OPN301-R - [REPEAT] Cloud Custodian: Open-source security and governance Cloud Custodian is a stateless rules engine with a YAML DSL that allows for policy-as-code infrastructure management that integrates with best-of-breed services and practices in AWS to provide for real-time policy enforcement. By composing actions and filters in a YAML DSL, Cloud Custodian provides an expressive syntax for solving problems across domains from security, cost optimization, governance, and operations. Its policies can integrate with Amazon GuardDuty, AWS Config, AWS Security Hub, AWS Lambda, AWS X-Ray, Amazon CloudWatch, and more. In this session, you learn how to write policies, deploy them across accounts, and monitor and manage their installation. View Less Workshop David Filiatrault Kapil Thangavelu
